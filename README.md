# membership-hub
Next-generation Multi-Tenant Academy &amp; Membership Operations Platform. High-scale, event-driven SaaS built with Quarkus, Kafka, PostgreSQL, Next.js (Web/Mobile Native via Capacitor), and GKE. Supports cross-tenant QR check-ins, omni-channel notifications (Zalo API, Push), Omni-Auth, and global SEO/Localization.
